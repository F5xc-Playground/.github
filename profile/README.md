## F5 XC Playground

Community tools and integrations for [F5 Distributed Cloud](https://www.f5.com/cloud). Built by F5 engineers solving their own problems — nothing official, everything useful.

---

### Projects

| Repository | Description |
|------------|-------------|
| [f5xc-k8s-operator](https://github.com/F5xc-Playground/f5xc-k8s-operator) | Kubernetes operator for managing F5 XC resources declaratively — load balancers, origin pools, WAF, and more as custom resources with automatic service discovery |
| [f5xc-namespace-backup](https://github.com/F5xc-Playground/f5xc-namespace-backup) | CLI tool to backup, restore, diff, and revert F5 XC namespace configurations with dependency-ordered operations across ~99 resource types |
| [f5xc-k8s-sigs-external-dns](https://github.com/F5xc-Playground/f5xc-k8s-sigs-external-dns) | ExternalDNS webhook provider for F5 Distributed Cloud DNS |
| [f5xc-console-plus-chrome](https://github.com/F5xc-Playground/f5xc-console-plus-chrome) | Chrome extension for the F5 XC Console — dark mode and visual tweaks |

---

### About

A place for F5ers to build on top of Distributed Cloud — the things you wish existed, the ideas you hacked on over a long weekend, the tools that make your day a little less annoying.

**What belongs here:** Tools, integrations, experiments, and proofs of concept that make working with XC better.

**What doesn't:** Customer data, internal secrets, or anything that should live in an official F5 org.

Want to add a project? Reach out to the org admins.
